# Cowboy Bebop Business Card App

## Overview
Created a business card as an android app inspired by the anime Cowboy Bebop.

## Features
- **XML Layout Mastery**: Proficient use of various XML layout elements including `ImageButton`, `ConstraintLayout`, `TabDots`, and `ViewPager`.
- **Kotlin Scripting**: Mastered basic Kotlin scripting with activities management, fragments, services, and adapters creation.
- **Anime Style Design**: The app is styled with a theme inspired by the anime series Cowboy Bebop.
